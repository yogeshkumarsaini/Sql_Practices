# Sql_Practices

Welcome to the **Sql_Practices** repository! This project contains a collection of SQL practice queries and exercises designed to help beginners and intermediates solidify their SQL skills. The repository covers a range of topics including basic CRUD operations, aggregate functions, joins, subqueries, and more.

---

## Introduction

SQL (Structured Query Language) is a standardized language used to manage, manipulate, and retrieve data from relational databases. This repository is intended as a helpful resource for practicing the most fundamental and commonly used SQL queries.

---

## Getting Started

- Clone this repo:  
  ```sh
  git clone https://github.com/<your-username>/Sql_Practices.git
  ```
- Import the provided `.sql` files into your preferred SQL database (MySQL, PostgreSQL, SQLite, etc.).
- Practice with the included queries, or modify them for deeper understanding.

---

## Topics Covered

### 1. Basic SQL

- Creating databases and tables
- Inserting data
- Updating data
- Deleting data
- Selecting data
- Filtering data (`WHERE`)
- Sorting results (`ORDER BY`)
- Limiting results (`LIMIT`)

### 2. SQL Functions

- Aggregate Functions: `COUNT()`, `SUM()`, `AVG()`, `MIN()`, `MAX()`
- String Functions: `CONCAT()`, `SUBSTRING()`, `LENGTH()`, `UPPER()`, `LOWER()`
- Date Functions: `NOW()`, `CURDATE()`, `DATEDIFF()`, `DATE_ADD()`
- Numeric Functions: `ROUND()`, `CEILING()`, `FLOOR()`

### 3. Joins

- Inner Join
- Left (Outer) Join
- Right (Outer) Join
- Full (Outer) Join
- Self Join
- Cross Join

### 4. Subqueries

- Subqueries in `SELECT`, `FROM`, and `WHERE` clauses
- Correlated subqueries
- Using subqueries with `IN`, `EXISTS`, `ANY`, `ALL`

### 5. Advanced Queries

- Grouping Data (`GROUP BY` and `HAVING`)
- Window Functions (if applicable)
- Case Statements (`CASE WHEN`)
- Common Table Expressions (CTEs) (if applicable)
